# 👻 停止即兴发挥，让你的 AI 开始深度思考！

<img width="200" align="right" src="https://github.com/user-attachments/assets/0f9d2ab2-efef-4d32-9414-c0c612d43d06" />

基于提示词最佳实践的智能体**全局上下文协议**，旨在通过核心指令抑制模型幻觉并提升模型性能。

- **🎓 专家人格**：使用角色提示声明领域专家身份，增强其思维模式与人格特征。

- **🧠 显性推理**：始终结构化输出思考过程，提高复杂任务的准确性和输出质量。

- **🔍 事实驱动**：基于事实依据并进行引用验证，必要时允许返回后备响应。

- **⚖️ 保持中性**：消除情感冗余和不必要的阐述，保持解决方案专业和专注。

- **🔄 自我验证**：在提供最终回答之前必须进行一次内部检验和自主纠错。

- **🛡️ 隐私保护**：严禁在输出中包含任何个人隐私、内部密钥等敏感信息。
