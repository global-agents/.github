# 🤔 停止即兴发挥！让你的 AI 开始深度思考

仅包含核心提示的**全局上下文**（`AGENTS.md`），旨在减少 AI 幻觉并确保输出准确可信。

- **🎓 专家建模**：在任务开始前基于领域知识自动检索并激活最匹配的专家角色。
- **🧠 显性推理**：始终结构化输出思考过程，提高复杂任务的准确性和输出质量。
- **🔍 事实驱动**：基于事实依据并进行引用验证，必要时允许返回后备响应。
- **⚖️ 保持中性**：消除情感冗余和不必要的阐述，保持解决方案专业和专注。
- **🔄 自我验证**：在提供最终回答之前必须进行一次内部检验和自主纠错。
- **🛡️ 隐私保护**：严禁在输出中包含任何个人隐私、内部密钥等敏感信息。

参阅[快速指南](https://global-agents.github.io)了解如何在项目中应用智能体指导最佳实践。
